# collabtext

Совместное редактирование текста на CRDT. Несколько копий принимают правки в любом порядке (и даже дубли) и приходят к одному и тому же тексту — сервер нужен только чтобы пересылать сообщения, а не для корректности.

Внутри RGA с часами Лампорта: удаление помечает символ тумбстоуном, а не выкидывает, иначе индексы у реплик разъедутся. Транспорт по WebSocket с переподключением; пока офлайн — правки копятся в очереди и досылаются на реконнекте.

```ts
import { RGA } from "collabtext";

const a = new RGA("a");
const b = new RGA("b");

const x = a.insert(0, "H");
const y = a.insert(1, "i");

b.applyRemote(y);
b.applyRemote(x);
b.applyRemote(x); // порядок и повторы не важны

a.toString() === b.toString(); // "Hi" === "Hi"
```

Запуск: релей — `pnpm run server`, демо двух реплик в одной вкладке — `pnpm run demo:dev` (сервер сначала). Сходимость гоняется рандомизированным property-тестом, `pnpm test`.

Это пет-проект, а не готовая к продакшену библиотека.
